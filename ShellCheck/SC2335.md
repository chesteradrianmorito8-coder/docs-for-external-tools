Pattern: Use of `[ ! "$var" -eq 1 ]`

Issue: -

## Description

Double negation of such binary operators is unnecessary.

Example of **incorrect** code:

```sh
if [ ! "$var" -eq 1 ]; then :; fi
if ! [ "$var" = foo ]; then :; fi
```

Example of **correct** code:

```sh
if [ "$var" -ne 1 ]; then :; fi
if [ "$var" != foo ]; then :; fi
```
